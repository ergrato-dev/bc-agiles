# Videografía — Semana 06

**Tema**: Artefactos de Scrum

---

### "What is a Product Backlog?" — Scrum.org
- **Plataforma**: YouTube (canal Scrum.org)
- **Duración**: ~10 minutos
- **Por qué**: Explica la diferencia entre Product Backlog, Sprint Backlog
  e Incremento con animaciones claras.

### "Definition of Done in Scrum" — Agile Institute
- **Plataforma**: YouTube
- **Duración**: ~12 minutos
- **Por qué**: Casos reales de equipos que no tenían DoD y los problemas
  que esto causó. Incluye cómo construir una DoD en equipo.

### "Writing Good User Stories" — Mountain Goat Software
- **Plataforma**: YouTube (Mike Cohn)
- **Duración**: ~20 minutos
- **Por qué**: Tutorial sobre User Stories con criterios INVEST, ejemplos
  de historias malas que se convierten en buenas paso a paso.
