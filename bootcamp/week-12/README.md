# Semana 12 — Métricas Ágiles: Velocity, Burndown y CFD

**Etapa 1: Scrum Practicante · Semana 12 de 24**

---

## ¿Qué aprenderás esta semana?

- Leer e interpretar un Burndown Chart para detectar riesgos en el Sprint
- Calcular y usar la Velocity del equipo para proyecciones futuras
- Interpretar un Cumulative Flow Diagram (CFD) para detectar cuellos de botella
- Diferenciar entre métricas de output (velocity) y métricas de outcome (valor entregado)
