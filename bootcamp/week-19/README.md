# Semana 19: DevOps y CI/CD en Contexto Ágil

## ¿Qué aprenderás esta semana?

El Manifiesto Ágil pide "software funcionando". DevOps y CI/CD son las
prácticas técnicas que hacen posible entregar ese software de forma continua,
segura y predecible.

## Objetivos

- Comprender la relación entre Agile, DevOps y la entrega continua
- Identificar las prácticas clave de CI/CD y cómo se integran al Sprint
- Reconocer prácticas de XP (Extreme Programming) complementarias a Scrum
- Distinguir los roles del Scrum Master y el equipo en la adopción de DevOps
