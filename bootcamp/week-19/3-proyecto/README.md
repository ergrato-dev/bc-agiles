# Proyecto Semana 19 — DoD con Criterios DevOps

## Contexto

Esta semana actualizas la Definition of Done de tu dominio para incluir criterios
de calidad técnica, CI/CD y prácticas XP.

## Objetivos del Proyecto

1. Revisar la DoD actual de tu producto (o crear una si no existe)
2. Añadir criterios de CI/CD y tests automatizados
3. Definir qué prácticas XP adoptará el equipo y cómo se integran al Sprint
