### Herramientas CI/CD populares

- **GitHub Actions**: CI/CD integrado en GitHub — gratuito para repos públicos
- **GitLab CI**: pipeline declarativo en `.gitlab-ci.yml`
- **Jenkins**: solución self-hosted, altamente configurable
- **CircleCI**: SaaS con capa gratuita generosa
