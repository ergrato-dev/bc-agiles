**Tema**: Sprint 1 de tu producto
**Etapa**: 0 — Fundamentos Ágiles

---

## Descripción

Este es el primer Sprint real de tu bootcamp. Deberás facilitar
(o simular) un Sprint completo usando el producto del dominio
que te asignó tu instructor.

Aplica TODOS los conceptos de las semanas 01–07:

- Manifiesto Ágil y principios (Week 01–02)
- Panorama de frameworks (Week 03)
- Roles de Scrum: SM, PO, Developers (Week 04)
- Eventos: Planning, Daily, Review, Retro (Week 05)
- Artefactos: Product Backlog, Sprint Backlog, Incremento (Week 06)
- Tablero Kanban dentro del Sprint (Week 07)
