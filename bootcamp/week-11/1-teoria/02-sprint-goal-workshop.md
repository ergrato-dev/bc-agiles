# 02 — Sprint Goal: El Objetivo que da Coherencia al Sprint

## Objetivo

Formular Sprint Goals orientados al valor del usuario, no a la lista de tareas.

---

## 1. ¿Qué es el Sprint Goal?

El Sprint Goal es el **único objetivo** del Sprint. Responde a:
*¿Qué valor recibirá el usuario al finalizar este Sprint?*

No es una lista de Stories. Es la razón por la que se hace ese Sprint.

---

## 2. Anatomía de un Sprint Goal efectivo

Un buen Sprint Goal tiene tres elementos implícitos:

| Elemento | Pregunta | Ejemplo |
| -------- | -------- | ------- |
| ¿Para quién? | ¿Qué usuario se beneficia? | "Los nuevos usuarios" |
| ¿Qué capacidad? | ¿Qué pueden hacer que antes no podían? | "puedan registrarse y completar su perfil" |
| ¿Para qué? | ¿Cuál es el valor de negocio? | "para reducir el abandono en onboarding" |

**Sprint Goal completo:**
> "Los nuevos usuarios pueden registrarse y completar su perfil básico,
> reduciendo el abandono en el proceso de onboarding."

---

## 3. Técnica: Sprint Goal Workshop (5 min)

Al final del Sprint Planning, el SM facilita:

1. **Cada persona escribe** en post-it: *"Este Sprint permite que [usuario] pueda [acción]"*
2. **Convergencia**: el equipo vota las 3 más parecidas
3. **Síntesis**: el PO redacta el Sprint Goal final con el input del equipo
4. **Validación**: ¿Todas las Stories seleccionadas contribuyen a este Goal?
   Si alguna no contribuye → revisar si realmente debe estar en el Sprint

---

## 4. Antipatrones del Sprint Goal

| Antipatrón | Problema | Corrección |
| ---------- | -------- | ---------- |
| "Implementar US-12, US-13 y US-14" | Es una lista, no un objetivo | Abstrae el valor que une esas Stories |
| "Mejorar la plataforma" | Demasiado vago, no medible | Agrega un criterio de éxito concreto |
| "Terminar todo el backlog" | Imposible de cumplir como objetivo único | Un Sprint = un enfoque de valor |
| Sprint Goal que cambia a mitad | Viola el principio de Sprint como caja de tiempo | El equipo puede rechazar cambios que afecten el Goal |

---

## Checklist

- [ ] ¿El Sprint Goal menciona un usuario o beneficiario?
- [ ] ¿Se puede verificar si el Goal se cumplió o no al final del Sprint?
- [ ] ¿Todas las Stories del Sprint contribuyen al mismo Goal?
- [ ] ¿El Goal fue creado con el equipo —no solo por el PO?
