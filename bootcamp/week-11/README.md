# Semana 11 — Sprint Planning Avanzado

**Etapa 1: Scrum Practicante · Semana 11 de 24**

---

## ¿Qué aprenderás esta semana?

- Calcular la capacidad real del equipo antes del Sprint Planning
- Formular un Sprint Goal orientado al valor del usuario
- Gestionar negociaciones entre el PO y el equipo de desarrollo
- Identificar compromisos insostenibles antes de que ocurran
