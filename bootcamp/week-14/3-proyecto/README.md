# Semana 14 — Proyecto Integrador: Sprint Review con Stakeholders

**Etapa 1: Scrum Practicante**
Aplica los conceptos de esta semana a tu dominio asignado.

---

## Objetivo

Facilitar y documentar un Sprint Review completo para tu producto,
incluyendo la demo del Incremento, captura de feedback de stakeholders
y actualización del Product Backlog.

---

## Escenario base

Tu equipo acaba de completar el **Sprint 5**. El Sprint Goal fue parcialmente alcanzado:
completaste el 75% de las historias comprometidas. Tienes 2–3 stakeholders clave
a quienes presentar los resultados.

Adapta el escenario a tu dominio asignado.
