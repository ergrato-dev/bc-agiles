# Semana 15 — Proyecto Integrador: Retrospectiva Completa

**Etapa 1: Scrum Practicante**
Aplica los conceptos de esta semana a tu dominio asignado.

---

## Objetivo

Facilitar y documentar una retrospectiva completa usando un formato avanzado,
con acciones SMART, seguimiento del Sprint anterior y manejo de al menos
una situación difícil.

---

## Escenario base

Tu equipo termina el **Sprint 5**. Tuviste resultados mixtos:
entregaste el 70% del Sprint Goal y tuviste al menos un incidente de calidad
o una dependencia no resuelta durante el Sprint.
