# Semana 13 — Gestión de Impedimentos y Deuda Técnica

**Etapa 1: Scrum Practicante · Semana 13 de 24**

---

## ¿Qué aprenderás esta semana?

- Distinguir entre un impedimento, un obstáculo y un problema del equipo
- Aplicar el proceso de escalación de impedimentos como Scrum Master
- Identificar y gestionar la deuda técnica en el backlog
- Negociar tiempo para pagar deuda técnica sin sacrificar la entrega de valor
