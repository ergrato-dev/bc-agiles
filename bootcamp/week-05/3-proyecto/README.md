# Proyecto — Semana 05: Diseña el Ciclo de Eventos

## Descripción

Esta semana diseñarás el ciclo de eventos de Scrum adaptado al producto
de tu dominio asignado. No solo definirás timeboxes: pensarás quién participa,
qué artefacto se produce y cómo facilitar cada evento para que agregue valor.

---

## Criterios de Evaluación

| Criterio | Peso |
|----------|------|
| Sprint Goal medible y orientado al usuario | 25% |
| Participantes correctos en cada evento | 25% |
| Timeboxes dentro de los límites de la Scrum Guide 2020 | 25% |
| Acciones de retrospectiva concretas (responsable + fecha) | 25% |
