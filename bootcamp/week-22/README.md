# Semana 22 — Facilitación Avanzada: Liberating Structures y Design Thinking

## ¿Qué aprenderás esta semana?

1. Aplicar al menos 5 Liberating Structures en eventos ágiles
2. Usar Design Thinking como proceso de innovación centrado en el usuario
3. Diseñar una sesión de facilitación avanzada con múltiples microestructuras
4. Distinguir cuándo usar Liberating Structures vs técnicas de retrospectiva clásicas
