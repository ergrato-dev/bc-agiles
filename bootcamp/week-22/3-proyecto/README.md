# Proyecto Semanal — Sesión de Facilitación Avanzada

**Semana 22 | Etapa 2: Ágil Avanzado**

---

## Objetivo del Proyecto

Diseñar y documentar una sesión de facilitación avanzada para tu equipo, usando al menos
3 Liberating Structures encadenadas o una sesión completa de Design Thinking aplicada a
un pain point real de tu producto.

---

## Instrucciones Generales

> ⚠️ **Adapta este proyecto a tu dominio asignado.**

Elige una de las dos opciones:

**Opción A**: Diseñar una Retrospectiva avanzada (60 min) con ≥3 LS encadenadas para
resolver un problema específico del equipo.

**Opción B**: Facilitar (o documentar) una mini-sesión de Design Thinking (90 min) para
descubrir y validar el pain point principal de un usuario de tu producto.

---

## Entregables

Ver `starter/entregables.md` para las plantillas a completar.

---

## Criterios de Evaluación

| Criterio | Peso |
|----------|------|
| Opcón A — Secuencia de 3 LS justificadas con diseño minuto a minuto | 100% |
| Opción B — 5 fases de DT con artefactos completos (mapa empatía, HMW, ideas, prototipo, criterio de test) | 100% |

**Aprobación mínima**: 70 puntos
