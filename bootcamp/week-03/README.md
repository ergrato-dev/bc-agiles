# Semana 03 — Panorama de Frameworks Ágiles

**Etapa 0 · Fundamentos Ágiles** | Semanas 1–8 de 24

## Objetivos de la semana

- Distinguir los principales frameworks ágiles: Scrum, Kanban, XP y SAFe
- Identificar en qué contexto conviene aplicar cada framework
- Reconocer que los frameworks son herramientas, no recetas
