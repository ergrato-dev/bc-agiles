# Proyecto Semanal — Selección y Justificación del Framework

**Semana 03 · Panorama de Frameworks Ágiles**

---

## Objetivo

Documentar la selección del framework ágil para tu producto/dominio asignado,
argumentando la decisión con análisis de contexto y anticipando los desafíos
de adopción.

---

## Entregables

Consultar `starter/entregables.md` para los TODOs detallados.

---

## Criterios de evaluación

| Criterio | Descripción |
| -------- | ----------- |
| Análisis contextual | 4 factores de decisión documentados con evidencia del dominio |
| Framework elegido | Nombre + variant (ej: Scrum, Kanban, Scrumban) |
| Justificación | Mínimo 3 razones específicas, no genéricas |
| Alternativas descartadas | Al menos 1 framework descartado con argumentos |
| Plan de adopción | 4+ semanas con acciones concretas y secuenciadas |
| Originalidad | Argumentos específicos al dominio asignado |
