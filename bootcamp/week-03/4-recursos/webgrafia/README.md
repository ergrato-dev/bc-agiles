**Tema**: Panorama de Frameworks Ágiles

---

## Referencias web

### Atlassian — "Kanban vs Scrum"
- **URL**: https://www.atlassian.com/agile/kanban/kanban-vs-scrum
- **Por qué**: Comparativa clara con diagramas, ejemplos y casos de uso.
  Orientada a practicantes, no a académicos.

### SAFe Framework — sitio oficial
- **URL**: https://scaledagileframework.com/
- **Por qué**: El referente obligado para SAFe. Artículos, diagramas y
  explicaciones de cada rol y evento del framework escalado.

### Agile Alliance — "Agile 101"
- **URL**: https://www.agilealliance.org/agile101/
- **Por qué**: Glosario y descripciones de todos los frameworks ágiles
  principales. Útil para comparar terminología entre Scrum, Kanban y XP.
