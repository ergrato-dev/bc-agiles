# Videografía — Semana 03

**Tema**: Panorama de Frameworks Ágiles

---

## Videos recomendados

### "Scrum vs Kanban" — Atlassian (serie de videos)
- **Plataforma**: YouTube (canal Atlassian)
- **Duración**: 5–10 minutos por video
- **Por qué**: Atlassian tiene una serie clara y concisa sobre diferencias
  entre Scrum y Kanban, con ejemplos de tableros visuales.

### "Introduction to SAFe" — Scaled Agile Inc.
- **Plataforma**: YouTube (canal Scaled Agile)
- **Duración**: ~15 minutos
- **Por qué**: Video oficial de introducción a SAFe. Explica los niveles
  del framework y qué problema resuelve en organizaciones grandes.

### "Extreme Programming Explained" — Kent Beck (charla)
- **Plataforma**: YouTube (búsqueda libre)
- **Duración**: ~45 minutos
- **Por qué**: Kent Beck, creador de XP, explica qué práctica de ingeniería
  distingue XP de cualquier otro framework. Esencial para entender TDD.
