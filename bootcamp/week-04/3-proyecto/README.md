# Proyecto Semanal — Roles en mi Dominio

**Semana 04 · Roles de Scrum**

---

## Objetivo

Adaptar los 3 roles de Scrum al contexto concreto de tu producto asignado.

## Entregables

Ver `starter/entregables.md` para los TODOs detallados.

## Criterios de evaluación

| Criterio | Descripción |
| -------- | ----------- |
| Especificidad | Cada rol se describe con ejemplos del dominio, no genéricos |
| Antipatrones | Cada rol incluye su antipatrón más probable en ese dominio |
| Interacciones | Describe fricciones reales esperadas entre los roles |
| Originalidad | Los ejemplos son propios del dominio asignado |
