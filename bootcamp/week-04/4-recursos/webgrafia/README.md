# Webgrafía — Semana 04

**Tema**: Roles de Scrum

---

### Scrum.org — Scrum Master Learning Path
- **URL**: https://www.scrum.org/pathway/scrum-master
- **Por qué**: Ruta oficial de aprendizaje del Scrum Master, con recursos
  gratuitos, assessments y artículos de la comunidad.

### Scrum.org — Product Owner Learning Path
- **URL**: https://www.scrum.org/pathway/product-owner-learning-path
- **Por qué**: Equivalente para el Product Owner. Incluye artículos sobre
  gestión del backlog, stakeholders y toma de decisiones.

### Mountain Goat Software — "Succeeding with Agile" (Mike Cohn)
- **URL**: https://www.mountaingoatsoftware.com/agile
- **Por qué**: Artículos de Mike Cohn sobre el rol del PO, estimación y
  relaciones con stakeholders. Especialmente útil para semana 9–11.
