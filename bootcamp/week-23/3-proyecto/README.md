# Proyecto Integrador — Semana 23

**Transformación Organizacional Ágil**

---

## Descripción

Este proyecto integra los dos marcos vistos esta semana (Kotter y Laloux) en un análisis completo de tu organización asignada. El resultado es un **Journey Map de Transformación**: un mapa del estado actual, las resistencias, las intervenciones y las métricas que indicarán que el cambio está ocurriendo.

---

## Dominio Asignado

Adapta el proyecto a tu dominio de negocio. Para ello, imagina que la empresa de tu dominio lleva 2 años en operación y acaba de decidir adoptar metodologías ágiles.

---

## Criterios de Evaluación

| Criterio | Peso |

| Diagnóstico Laloux (nivel actual justificado con comportamientos) | 30% |
