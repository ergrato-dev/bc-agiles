# Proyecto Semana 18 — Tablero Kanban Avanzado

## Contexto

Esta semana diseñas un tablero Kanban avanzado para tu dominio asignado,
con políticas explícitas, clases de servicio y métricas de flujo reales.

## Objetivos del Proyecto

1. Definir las columnas del tablero con WIP limits justificados
2. Establecer políticas de entrada y salida para cada columna
3. Definir 3 clases de servicio adaptadas a tu dominio
4. Simular 2 semanas de datos y calcular Lead Time y Throughput
