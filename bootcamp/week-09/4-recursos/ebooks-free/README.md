# Recursos — Ebooks Gratuitos

**Semana 09 · User Stories**

## User Stories Applied — Mike Cohn (resumen gratuito)
- **Tipo**: Resumen oficial en web
- **Enlace**: https://www.mountaingoatsoftware.com/books/user-stories-applied
- **Por qué leerlo**: El libro de referencia sobre User Stories. El resumen
  en la web cubre los capítulos fundamentales de INVEST y criterios de aceptación.

## Fifty Quick Ideas to Improve Your User Stories — Gojko Adzic
- **Tipo**: Fragmento gratuito (Amazon preview / Leanpub)
- **Enlace**: https://leanpub.com/50quickideas
- **Por qué leerlo**: 50 ideas prácticas para escribir mejores Stories.
  Leer los primeros 10 capítulos es suficiente para esta semana.

## Scrum.org — Backlog Refinement Explained
- **Tipo**: Artículo oficial gratuito
