# Semana 07 — Introducción a Kanban

## ¿Qué aprenderás esta semana?

Kanban no es solo un tablero de post-its. Es un sistema de gestión de
flujo de trabajo basado en visualización, limitación del WIP y mejora
continua. Esta semana aprenderás los principios y prácticas básicas
que hacen de Kanban una herramienta poderosa para equipos de servicio
y soporte.

---

## Objetivos de Aprendizaje

- Configurar un tablero Kanban funcional con columnas y WIP limits
- Identificar y medir el Lead Time y el Cycle Time
- Aplicar las 6 prácticas básicas de Kanban
- Distinguir cuándo usar Kanban vs Scrum vs Scrumban
