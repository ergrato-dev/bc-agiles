# Proyecto — Semana 07: Tablero Kanban para tu Producto

## Descripción

Diseñarás un tablero Kanban real para tu dominio asignado. No es solo poner
columnas: definirás el flujo real, los WIP limits justificados y al menos
una política explícita de columna.

---

## Criterios de Evaluación

| Criterio | Peso |
|----------|------|
| Columnas reflejan el flujo real del dominio | 25% |
| WIP limits con justificación documentada | 25% |
| Al menos 1 política explícita de columna | 25% |
| 5 ítems de trabajo representativos del dominio en el tablero | 25% |
