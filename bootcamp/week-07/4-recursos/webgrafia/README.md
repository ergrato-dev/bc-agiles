# Webgrafía — Semana 07

**Tema**: Introducción a Kanban

---

### Atlassian — "What is Kanban?"
- **URL**: https://www.atlassian.com/agile/kanban
- **Por qué**: Explicación accesible con ejemplos visuales, historia de
  Kanban y guía de implementación básica con Jira Boards.

### Kanban University — "Kanban Practices"
- **URL**: https://kanban.university/kanban-guide/#kanban-practices
- **Por qué**: Las 6 prácticas de Kanban con ejemplos concretos. Es la
  sección más práctica de la guía oficial.

### LeanKit — "Lead Time vs. Cycle Time"
- **URL**: https://leankit.com/learn/kanban/kanban-metrics/
- **Por qué**: Artículo con gráficos explicativos de Lead Time vs Cycle
  Time, incluyendo ejemplos de cálculo en tableros reales.
