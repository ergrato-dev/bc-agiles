# Proyecto Semana 01 — Define tu Producto

## Descripción

Antes de aplicar cualquier marco ágil necesitas conocer profundamente
el producto sobre el que trabajarás durante las 24 semanas del bootcamp.
En este proyecto definirás la visión inicial de tu producto y crearás
tus primeras User Stories.

## Instrucciones

1. Revisa el dominio de negocio que te fue asignado por el instructor
2. Completa el archivo `starter/entregables.md` con la información de tu producto
3. Redacta la Product Vision y las primeras User Stories

> Adapta las instrucciones a tu dominio asignado:
>
> - App delivery → pedidos, repartidores, restaurantes
> - E-commerce → productos, carrito, compradores
> - App bancaria → cuentas, transferencias, usuarios
> - Plataforma educativa → cursos, estudiantes, instructores

## Entregable

Completa el archivo `starter/entregables.md` con tu implementación.

## Criterio de éxito

- La Product Vision identifica claramente quién es el usuario, qué necesita
  y cómo se diferencia de la alternativa actual
- Las User Stories siguen el formato Como/Quiero/Para
- El trabajo refleja el dominio asignado, no el ejemplo del starter
