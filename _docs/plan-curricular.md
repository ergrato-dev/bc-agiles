# 📋 Plan Curricular — Bootcamp Ágil de Cero a Héroe

**Duración**: 24 semanas (~6 meses)  
**Dedicación**: 8 horas semanales  
**Total**: ~192 horas  
**Nivel de entrada**: Cero (sin experiencia previa en metodologías ágiles)  
**Nivel de salida**: Scrum Master Junior / Product Owner Junior / Agile Coach Junior  
**Herramienta principal**: GitHub Projects + Markdown (artefactos y simulaciones)

---

## Etapa 0 — Fundamentos Ágiles · Semanas 1–8 · 64h

| Semana | Título                               | Temas clave                                                                              | Horas |
| ------ | ------------------------------------ | ---------------------------------------------------------------------------------------- | ----- |
| 01     | Del Cascada al Ágil                  | Crisis del software, modelo en cascada, Manifiesto Ágil (2001), los 4 valores            | 8h    |
| 02     | Los 12 Principios Ágiles             | Análisis de cada principio, aplicación en equipos reales, antipatrones frecuentes        | 8h    |
| 03     | Panorama de Frameworks Ágiles        | Scrum, Kanban, XP, Scrumban, SAFe: características y cuándo usar cada uno                | 8h    |
| 04     | Roles de Scrum                       | Scrum Master, Product Owner, Equipo de Desarrollo: responsabilidades, diferencias y DoR  | 8h    |
| 05     | Eventos de Scrum                     | Sprint, Sprint Planning, Daily Scrum, Sprint Review, Retrospectiva: timeboxes y propósitos | 8h  |
| 06     | Artefactos de Scrum                  | Product Backlog, Sprint Backlog, Incremento, Definition of Done, Definition of Ready     | 8h    |
| 07     | Introducción a Kanban                | Tablero Kanban, columnas, WIP limits, flujo de trabajo, comparativa Scrum vs Kanban      | 8h    |
| 08     | Proyecto Integrador Etapa 0          | Simulación completa: roles + eventos + artefactos básicos en un Sprint de práctica       | 8h    |

---

## Etapa 1 — Scrum Practicante · Semanas 9–16 · 64h

| Semana | Título                               | Temas clave                                                                              | Horas |
| ------ | ------------------------------------ | ---------------------------------------------------------------------------------------- | ----- |
| 09     | User Stories                         | Formato Como/Quiero/Para, criterio INVEST, criterios de aceptación en Gherkin            | 8h    |
| 10     | Jerarquía del Backlog                | Épicas, Features, Stories, Tasks: descomposición vertical, Backlog Refinement            | 8h    |
| 11     | Estimación Ágil                      | Planning Poker, T-Shirt Sizing, Fibonacci, #NoEstimates, Relative Sizing                 | 8h    |
| 12     | Sprint Planning                      | Sprint Goal, capacidad del equipo, descomposición de backlog en tareas, Definition of Ready | 8h |
| 13     | Métricas Ágiles I                    | Velocity, Burndown Chart, Burnup Chart: cálculo, interpretación y predicción              | 8h    |
| 14     | Métricas Ágiles II + Impedimentos    | CFD, cycle time, lead time, Cumulative Flow Diagram, gestión y escalación de impedimentos | 8h    |
| 15     | Sprint Review y Demo                 | Facilitación efectiva, inspección del Incremento, feedback de stakeholders, DoD          | 8h    |
| 16     | Retrospectivas                       | 4Ls, Start/Stop/Continue, Fishbone, Mad/Sad/Glad, acciones Kaizen con seguimiento        | 8h    |

## Etapa 2 — Ágil Avanzado · Semanas 17–24 · 64h

| Semana | Título                               | Temas clave                                                                              | Horas |
| ------ | ------------------------------------ | ---------------------------------------------------------------------------------------- | ----- |
| 17     | Scaled Agile I — SAFe                | Niveles Team/Program/Large Solution/Portfolio, PI Planning, Agile Release Train          | 8h    |
| 18     | Scaled Agile II — LeSS y Nexus       | LeSS vs Nexus vs SAFe: cuándo usar cada uno, Sprint de integración, Scrum of Scrums      | 8h    |
| 19     | Kanban Avanzado                      | Políticas explícitas, clases de servicio, cadencias, métricas de flujo, Kanban Maturity  | 8h    |
| 20     | DevOps y CI/CD en Contexto Ágil      | Integración dev-ops, entrega continua, DoD en pipelines, Definition of Done expandida    | 8h    |
| 21     | OKRs y Alineación Estratégica        | Objectives & Key Results, alineación con Product Backlog, roadmap de producto, bets      | 8h    |
| 22     | Agile Coaching                       | Niveles de coaching, estilos de conversación, stance del Agile Coach, shu-ha-ri          | 8h    |
| 23     | Facilitación Avanzada y Transformación | Liberating Structures, Design Thinking, gestión del cambio, journey map organizacional  | 8h    |
| 24     | Proyecto Final — Capstone            | Simulación completa con Scaled Agile, presentación de artefactos y preparación PSM I     | 8h    |

## Herramienta Principal por Etapa

| Etapa   | Semanas | Herramienta principal                                           |
| ------- | ------- | --------------------------------------------------------------- |
| Etapa 0 | 1–8     | GitHub Projects (Board básico) + Markdown                       |
| Etapa 1 | 9–16    | GitHub Projects (avanzado: labels, milestones, sprints) + Miro  |
| Etapa 2 | 17–24   | GitHub Projects + Miro / FigJam + plantillas SAFe/LeSS          |

> ⚠️ La incorporación de Miro/FigJam ocurre en la semana 9, con la
> introducción de User Stories y estimación colaborativa. A partir de
> esa semana las retrospectivas y los Planning Poker se simulan en
> tableros digitales colaborativos.
